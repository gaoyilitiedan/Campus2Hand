# Campus2Hand 后端实现文档

## 一、任务完成情况总览

| 阶段 | 任务 | 状态 | 产出物 |
|------|------|------|--------|
| Phase 0 | Maven 多模块项目结构 + 父 POM + 配置文件 | ✅ 完成 | 13 个 POM + application.yml |
| Phase 1 | common 模块（ApiResponse/ErrorCode/异常/工具类/配置） | ✅ 完成 | 9 个 Java 文件 |
| Phase 2 | 数据库 Flyway 迁移脚本（15 张表 DDL + 种子数据） | ✅ 完成 | V1__init_schema.sql + V2__seed_data.sql |
| Phase 3 | auth 模块（注册/登录/Refresh/改密码/邮箱验证） | ✅ 完成 | 17 个 Java 文件 |
| Phase 4 | user 模块（个人信息/手机号绑定/信誉分查询） | ✅ 完成 | 4 个 Java 文件 |
| Phase 5 | product 模块（搜索/详情/发布/管理/图片上传/收藏） | ✅ 完成 | 12 个 Java 文件 |
| Phase 6 | order 模块（下单/支付/确认收货/退款/状态机） | ✅ 完成 | 7 个 Java 文件 |
| Phase 7 | chat 模块（会话/消息/已读/未读计数） | ✅ 完成 | 9 个 Java 文件 |
| Phase 8 | review + dispute + notification + admin 模块 | ✅ 完成 | 22 个 Java 文件 |
| Phase 9 | job 定时任务模块 + 启动入口 | ✅ 完成 | 2 个 Java 文件 |

**总计：12 个模块，80+ 个 Java 源文件，2 个 SQL 迁移脚本，13 个 POM 文件**

## 九、启动说明

### 前置条件

1. JDK 17+
2. Maven 3.8+
3. MySQL 8.0+（创建数据库 `campus2hand`）
4. Redis 7.0+（可选，当前未强依赖）

### 启动步骤

```bash
# 1. 进入项目目录
cd campus2hand-server

# 2. 编译项目
mvn clean compile -DskipTests

# 3. 启动应用（Flyway 会自动执行数据库迁移）
mvn spring-boot:run -pl campus2hand-server

# 4. 访问 Swagger UI
# http://localhost:8080/swagger-ui.html
```

### 数据库初始化

Flyway 会在首次启动时自动执行：
- `V1__init_schema.sql`：创建全部 15 张表
- `V2__seed_data.sql`：插入敏感词库 + 管理员账号（admin001 / admin@campus2hand.com）
